## 🚀 快速开始

### 添加新知识

1. **技术笔记**: 放到 `01-技术知识/对应分类/`
2. **行业方案**: 放到 `02-商业变现/行业方案/行业名/`
3. **文章草稿**: 放到 `03-内容创作/对应平台/`

### 命名规范

- 文件: `YYYY-MM-DD-标题.md`
- 图片: `assets/图片描述.png`
- 模板: `_template-类型.md`
