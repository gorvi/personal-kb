## 🛠️ 技术栈建议

### 小程序开发（推荐）
```
框架：uni-app（一套代码，多端运行）
UI：uView / ThorUI
后端：云开发（微信云开发/阿里云）
AI辅助：Cursor + GPT-4o / Claude
```

### 官网建站（快速交付）
```
工具：Webflow / Framer
模板：Envato Elements
部署：Vercel / Netlify
```

### SaaS开发（后期）
```
前端：Next.js + TailwindCSS
后端：Supabase / Firebase
支付：Stripe / 微信支付
部署：Vercel + Railway
```
