# 语言培训机构建站与经营全攻略

## 一、行业痛点分析

1. **师资展示需求**：需要展示外教/中教资质
2. **课程体系复杂**：需要清晰展示分级教学
3. **学习效果证明**：需要展示学员提分案例
4. **试听转化**：需要降低试听门槛

---

## 二、网站核心功能模块

```
首页
├── 师资团队（外教+中教介绍）
├── 课程体系（幼儿/小学/中学/成人）
├── 分级测试（在线测试英语水平）
├── 学员案例（提分案例+口语展示）
├── 在线试听（预约试听课）
├── 考试资讯（雅思/托福/四六级信息）
└── 学习资源（免费资料下载）
```

## 四、获客渠道

1. **学校周边**： targeting 学生群体
