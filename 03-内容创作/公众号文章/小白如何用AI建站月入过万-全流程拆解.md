## 一、为什么要做建站生意？

### 1. 市场需求大

你算一下，中国有多少小微企业？

- 餐饮店：每个城市几千家
- 美容美发：每个城市几千家
- 教培机构：每个城市几百家
- ...

**99%的小微企业都需要一个网站**，但只有1%有。

这是一个巨大的市场空白。

### 2. 门槛变低了

以前做网站：
- 需要会代码（HTML/CSS/JS）
- 需要懂设计（UI/UX）
- 需要会部署（服务器/域名）

现在做网站：
- **AI工具一键生成**
- 模板直接套用
- 可视化编辑

**小白也能做**。

### 3. 客单价不错

一个网站的收费（2026年市场价）：
- 基础版：999-1499元
- 标准版：1999-2999元
- 进阶版：3999-5999元

**一个月做10单，收入就是1万-6万**。

> 为什么能收这个价？
> - 2026年AI建站效率高，但客户买的是「解决方案」不是「代码」
> - 你省下的时间，应该用来提升服务质量，而不是降价
> - 定价太低反而没利润，也无法持续提供服务

### 4. 复购率高

客户做完网站后，还会有：
- 内容更新（每月收维护费）
- 功能升级（加功能另外收费）
- 转介绍（客户介绍朋友来）

**一次成交，长期收益**。

## 三、完整工作流程

### Step 1：找客户（最关键）

#### 方法一：平台获客

**美团/大众点评**：
1. 搜"餐饮店"+"城市名"
2. 筛选评分4.0以上的店
3. 点进去看有没有网站
4. 没网站的，记录下来
5. 批量联系

**小红书**：
1. 搜"城市名+行业"（如"北京美容"）
2. 找粉丝少但作品好的店
3. 看主页有没有网站链接
4. 没网站的，私信联系

**抖音本地生活**：
1. 看本地商家的抖音号
2. 看主页有没有官网
3. 没官网的，评论区留言

#### 方法二：地推

**美食街/商场**：
1. 准备宣传单（写清楚服务内容+价格）
2. 进店找老板聊
3. 话术：
   > "老板，我看您店生意不错，但很多顾客搜您店名时，可能搜到的是竞争对手。有个自己的网站，就能牢牢占据品牌搜索的第一入口。"

#### 方法三：转介绍

**给老客户好处**：
- 介绍1个客户，送1个月维护
- 介绍3个客户，返现500元
- 介绍5个客户，免费升级一个功能

**我的经验**：转介绍的客户成交率最高，因为他们已经信任你了。

## 四、定价策略（2026年最新）

### 为什么2026年要涨价？

很多新手不敢报价，还在用2023年的价格：基础版599、标准版999。

但2026年的情况是：
- **AI工具普及了**——做网站效率提升3倍，但客户对"专业度"要求更高了
- **竞争分化**——低价抢单的人太多，反而中高端市场有空缺
- **客户认知升级**——现在老板都知道网站重要，愿意为效果付费

**你卖的不是代码，是获客解决方案。**

### 我的定价表（2026年执行版）

| 版本 | 价格 | 适合谁 | 交付内容 | 制作周期 |
|------|------|--------|----------|----------|
| **体验版** | ¥999 | 预算极低，试水 | 单页展示 | 1-2天 |
| **基础版** | ¥1,499 | 小微企业 | 3-5页 | 3-5天 |
| **标准版** | ¥2,499 | 主流客户（80%选这个） | 5-10页+功能 | 7-10天 |

| **定制版** | ¥5,999+ | 特殊需求 | 定制开发 | 按项目 |

### 定价心理学技巧

**技巧1：锚定效应**
```
"我们做过最贵的一个网站是1万多，是给连锁品牌做的。像您这种需求，标准版2499就够用了。"
```

**技巧2：套餐对比（助推中间选项）**
```
体验版：999元（单页展示，功能简单）
标准版：2499元（5页+在线预约+会员系统）⭐ 推荐
进阶版：3999元（10页+全部功能+优先服务）
```

**技巧3：限时赠品（不降价）**
```
"这个月下单，送：
- 价值500元的百度地图标注
- 价值300元的首年域名
- 价值200元的LOGO设计
总共省了1000块，相当于2499享受了3499的服务。"
```

**技巧4：风险逆转**
```
"您放心，我们先收50%定金，做完您满意了再付尾款。如果7天内有不满意的，免费修改到满意为止。"
```

## 六、常见问题

### Q1：我不会设计怎么办？

**答**：
1. 用模板（Wix/Canva有大量模板）
2. 用AI生成（Midjourney/DALL-E）
3. 找设计师合作（外包设计，你做技术）

### Q2：客户要我开发复杂功能怎么办？

**答**：
1. 先评估能不能做（用现有工具能不能实现）
2. 能做就接，不能做就拒绝
3. 或者找外包（你赚差价）

**我的原则**：
- 用现成工具能做的 → 接
- 需要写代码的 → 不接（或者外包）

### Q3：客户砍价怎么办？

**答**：
1. **价值锚定**：强调网站的价值（获客、品牌）
2. **套餐对比**：给3个版本，让他选中间
3. **限时优惠**："这个月有活动，送XXX"
4. **分期付款**："可以分两期付"
5. **如果实在要砍**：功能相应减少

### Q4：如何保证质量？

**答**：
1. **标准化流程**：每个项目都按流程做
2. **检查清单**：交付前逐项检查
3. **客户确认**：关键节点让客户确认
4. **及时沟通**：有问题及时和客户说

### Q5：如何处理客户修改意见？

**答**：
1. **合同约定**：明确修改次数（如3次免费）
2. **超出收费**：超出次数按小时收费
3. **引导决策**：给专业建议，不要客户说啥做啥
4. **留痕确认**：每次修改后让客户确认
